Homepage Dissector Release Text

This release introduces a new feature that will make on-boarding and support easier for LocalFusion sites! This is the ability to get more information about a certain publisher by simply adding the following parameter to the end of any new platform URL:

?dissector=true

Walkthrough

This is how a platform homepage looks normally:

If we add the ?dissector=true parameter, we now see a sidebar on the left side. This gives us a bunch of handy information with links to settings:

It also displays the map's zoom controls and tracks any latitude, longitude and zoom changes:

This is super handy because, at this moment, it's a hassle to determine the best lat/lon/zoom levels for each publisher. By knowing the exact values needed in the MP Admin settings, on-boarding will have a much easier time getting a site looking at its best.

This feature will only work on homepages at this moment.
